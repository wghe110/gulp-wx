#gulp 移动端开发模板

####需要环境
- nodeJs
- ruby
- sass
- git
- gulp

*使用前先配置里面相关的参数config_ftp*

####安装依赖

*最好使用淘宝的node镜像和ruby镜像，天朝的网速。。。你懂的*

```javascript
cnpm install
```

####命令

gulp server  开启本地服务器

gulp dist    生成dist压缩版本

gulp upload  上传到服务器

*LongPathTool_v2.20.exe是用来删除node_modules目录的，便于项目保存，手动删除不掉它的*
